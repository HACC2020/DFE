Our site is meant to provide meaningful data and visualizations about the projects and applications on the states IT Roadmap that are easily available to the public.

Overall metrics for the entire IT portfolio are displayed above, separated by applications and projects. Applications are actual software systems underneath each respective department and the projects are associated with how the work to update the system is done. An application/project can be defined as “completed” when the department that owns it fills out all the data columns that are provided by ETS. If any data column fails to be completed, that application/project automatically becomes flagged as uncompleted. {% endblock %}
